With registration

In this version of Notepad you need to register a personal account using e-mail. This will allow you to log into your account from multiple devices and synchronize all the information in a single application. Your data will be password protected and stored on the server cloud. To work in this version, you must have a mandatory connection to the Internet.

Without registration

This version of Notepad does not require registration of a personal account, since it is intended for use on only one device. All information is stored right in your browser and is not deleted when it is closed. Work in the application can be done without connecting to the Internet.
